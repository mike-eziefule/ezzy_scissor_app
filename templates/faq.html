FAQS

How do URL shortening work?

URL shortener works by taking a long URL and creating a shorter, condensed version that directs to the original URL. when a user clicks on the link, they are redirected to the intended destination.

Is it necessary to create an account to use the URL shortening service?

Yes, Ez-Scissor provides free services like URL storage, QR code generation, Domain customization, and analysis therefore users are required to create an account to enable access to a dashboard where these services can be organized.

Are the shortened links permanent? Will they expire?

Shortened URLs are permanent except deleted by owner.

Are there any limitations on the number of URLs I can shorten?

In version 0.1.0, No limitations has been set however, users a restricted to 10 shortening per minute.

Can I customize the shortened URLs to reflect my brand or content?

Yes you can, create a free account and explore this possibilities.

Can I track the performance of my shortened URLs?

Analytics for all created short links are available on the dashboard.

How secure is the URL shortening service? Are the shortened links protected against spam or malicious activity?

Shortened links can only be modified by registered user alone. Endevour to create a strong password for your account.

What is a QR code and what can it do?

QR Code are images containing information of the qr link created. These images can be scanned and decodes by any qr scanner.

Is there an API available for integrating the URL shortening service into my own applications or websites?

None for now, this will be integrated in the future.
